# ⚽ Football Analyzer (战神卡深度分析工具)

这是一个专为 VS Code **Claude Code** 插件设计的足球比赛分析 Skill。

### 🌟 核心特色
- **专家逻辑**：内置 15 年资深足球分析专家“战神卡”的复盘思维。
- **深度叙事**：拒绝表面胜负，从市场心理、指数维稳、天然削弱等维度进行深度拆解。
- **合规表达**：自动过滤敏感词汇，输出专业、平铺、可直接用于复盘的文案。

### 🚀 如何安装到你的 VS Code
在 Claude Code 的对话终端中输入以下指令：
import github.com/liaomang929/football-analyzer

### 📝 使用方法
调用时提供两个变量：
1. **match**: 比赛对阵及基本面（如：利物浦vs巴黎，主帅下课）。
2. **odds**: 市场数据（如：平手低返）。
