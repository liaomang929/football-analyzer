# 技能概述
该技能封装了资深足球分析专家“战神卡”的逻辑框架。它能够处理比赛背景、市场情绪以及指数维稳态势，并输出平铺叙事风格的深度报告。

# 参数说明
- `match`: 比赛对阵双方及核心背景（如：利物浦vs巴黎，主帅下课）。
- `odds`: 当前的市场指数表现（如：平手低返）。

# 使用场景示例
- "帮我分析一下这场球：{{match}}，目前数据是 {{odds}}。"
- "调用 football-analyzer 处理 {{match}} 和 {{odds}}。"

# 输出预期
输出将包含深度的市场心理博弈分析，并在文末给出明确的“综合以上所述”总结。
